Omnes MarketPlace – Projet Web APP 2026
Description

Omnes MarketPlace est une plateforme e-commerce développée dans le cadre du Projet Programmation Web APP 2026.
Le site permet aux utilisateurs d’acheter et de vendre des articles en ligne via plusieurs modes de vente : achat immédiat, négociation entre vendeur et acheteur, ou enchères (meilleure offre).

L’objectif du projet est de développer une application web complète comprenant un front-end et un back-end, ainsi qu’une base de données permettant la gestion des utilisateurs, des articles et des transactions.
